## 🎮 How to Play  
1. Control the snake using these keys:  
   - **W**: Move Up ⬆️  
   - **A**: Move Left ⬅️  
   - **S**: Move Down ⬇️  
   - **D**: Move Right ➡️  

2. **Objective**:  
   - Eat food to score points.  
   - Avoid colliding with the walls or your own tail!
